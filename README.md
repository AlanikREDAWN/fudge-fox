# Fudge Fox

Fudge Fox is a game about helping a little fox (who really likes fudge) eat all the fudge in the world! 

## How to Play

Goal: Find and collect the fudge to open the door

## Controls:

**Spacebar:** Jump

**A/Left Arrow:** Move left

**D/Right Arrow:** Move right

**W/Up Arrow:** Climb up (when on ladder)

**S/Down Arrow:** Climb down (when on ladder)
